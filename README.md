🧠 Amygdalé — Financial Dashboard & Risk Control

Amygdalé (del griego ἀμυγδαλή, amígdala) es un dashboard financiero personal diseñado bajo la filosofía de "Security-First" y "Second Brain". Así como la amígdala procesa las emociones y el sistema de recompensa, esta aplicación centraliza el control de tus finanzas para facilitar la toma de decisiones lógicas frente a los impulsos económicos.

🔗 Enlaces del Proyecto
Live Demo: https://amygdale.netlify.app/

Repositorio: https://github.com/Neymors/Amygdale

🚀 Propósito del Proyecto
Este dashboard forma parte del ecosistema Enképhalos. Su objetivo es proporcionar una interfaz limpia y segura para el seguimiento de activos, gastos y salud financiera, integrándose eventualmente como un módulo modular (vía Iframe o API) dentro de Ippókampos.

🛠️ Stack Tecnológico

Frontend: React para una interfaz dinámica y reactiva.


Estilos: Tailwind CSS para un diseño minimalista y profesional.


Animaciones: Framer Motion para transiciones fluidas en el dashboard.


Despliegue: Netlify para la versión web.

📊 Funcionalidades (Roadmap)
Fase 1: Visualización: Gráficos de ingresos vs. egresos y balance general.


Fase 2: Integración: Conexión segura para servir datos a Ippókampos.

Fase 3: Seguridad: Implementación de cifrado local para proteger los montos y datos sensibles de la vista pública.

🔐 Seguridad y Privacidad
Siguiendo la línea de tus otros proyectos como Thálamos y CryptoGen:

Local-First Focus: Prioridad en la persistencia de datos del lado del cliente.


Identidad Criptográfica: Preparado para integrarse en tu Hub central con una identidad única.

📂 Cómo ejecutar el proyecto localmente
Clona el repositorio:

Bash
git clone https://github.com/Neymors/Amygdale.git
Instala las dependencias:

Bash
npm install
Inicia el servidor de desarrollo:

Bash
npm run dev
